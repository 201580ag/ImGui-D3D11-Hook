**폰트 적용 방법**
```
1. `HxD` 를설치한다.
2. 원하는 폰트를 다운 받는다.(.ttf)
3. HxD 에 폰트(.ttf) 파일을 넣는다.
4. 그리고 저장 또는 클립보드 복사를 C 형식으로 한다.
5. 복사나 저장한 C로 변환된 폰트를 `font.h` 안에 넣는다.
```
